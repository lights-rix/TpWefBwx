# 项目简介

## 核心功能
- 基于YOLOv8的猫行为检测系统，可识别6类猫行为：eating(进食)、playing(玩耍)、rest(休息)、sitting(坐着)、stretching(伸展)、yawning(打哈欠)
- 支持多种输入源检测：单张图片、图片文件夹、视频文件、本地摄像头
- 提供用户界面，实时显示目标位置、目标总数、置信度等信息
- 支持自定义检测参数：置信度阈值、IOU阈值
- 支持选择显示标签和原图
- 支持选择不同检测模型
- 支持批量检测并查看所有检测结果
- 支持保存检测结果

## 适用场景
- 猫咪行为研究与分析
- 宠物行为监测
- 动物行为学研究
- 家庭宠物活动记录

## 技术特点
- 基于深度学习YOLOv8模型开发
- 使用Python3.9、OpenCV、PyQt5、Ultralytics、Torch1.9等技术栈
- 提供完整的源码、标注数据集、训练好的模型及训练结果
- 包含5000张标注数据集
- 提供详细的项目运行教程
- 配置成功环境可直接使用
